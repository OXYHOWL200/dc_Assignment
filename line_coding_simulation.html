Aim of the Experiment

To study and analyze various line coding schemes used in digital communication systems including Unipolar NRZ, Unipolar RZ, Polar NRZ, Polar RZ, Bipolar AMI, Manchester, Duo Binary, and HDB3 encoding techniques.

The experiment aims to help students understand the characteristics, advantages, and applications of different line coding techniques and visualize their waveforms using a virtual oscilloscope.

Theory

Line Coding: Line coding is the process of converting digital data to digital signals. It involves representing binary data (0s and 1s) using various voltage levels and transitions.

Types of Line Coding Schemes:

1. Unipolar NRZ: Uses only positive voltage. Logic 1 = +V, Logic 0 = 0V. No return to zero between bits.

2. Unipolar RZ: Similar to Unipolar NRZ but signal returns to zero in the middle of each bit period.

3. Polar NRZ: Uses both positive and negative voltages. Logic 1 = +V, Logic 0 = -V.

4. Polar RZ: Similar to Polar NRZ but returns to zero in the middle of each bit.

5. Bipolar AMI: Logic 0 = 0V, Logic 1 alternates between +V and -V.

6. Manchester: Logic 1 has transition from low to high, Logic 0 has transition from high to low in the middle of bit period.

7. Duo Binary: Uses correlation between adjacent bits for encoding.

8. HDB3: High Density Bipolar of order 3, prevents long sequences of zeros.

Observations:

1. Unipolar NRZ uses only positive voltage levels.

2. Polar schemes provide better noise immunity due to voltage symmetry.

3. Manchester encoding has guaranteed transitions for clock recovery.

4. Bipolar AMI and HDB3 have zero DC component.
